# CV-CHESS
This is a script that detects moves on a chessboard and then sends the moves 
to a chess engine (Stockfih) in order to calculate the best move. ╰(*°▽°*)╯

## Requirements
 * Python 3
 * Open Cv librarys
 * Stockfish engine
 * Some others librarys
